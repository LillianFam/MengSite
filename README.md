# Meng Tian - Personal Website

This is a personal portfolio website for Meng Tian, showcasing her education, certifications, and contact information.

## How to View

Simply open the `index.html` file in any modern web browser (Chrome, Firefox, Edge, Safari).

## Technologies Used

- **HTML5**: Semantic structure.
- **Tailwind CSS**: Utility-first CSS framework (loaded via CDN) for styling.
- **Google Fonts**: Playfair Display (Headings) and Lato (Body) for typography.
- **JavaScript**: Simple interactivity for the mobile menu and smooth scrolling.

## Editing

To make changes, you can edit the `index.html` file directly in any text editor.
